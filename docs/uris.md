Web Console
: \<baseURL>/
: Content
    : \<baseURL>/c/<content_id>

Ontology Namespace
: \<baseURL>/ns/<ontology_id>

Data Management System
: \<baseUrl>/dms

Data Storage Management Service
: \<baseUrl>/dms/dsms
: Data Storage System
    : \<baseUrl>/dms/dss/<storage_service_id>
    : Minio
        : \<baseUrl>/dms/dss/minio
    : PostgreSQL
        : \<baseUrl>/dms/dss/pgsql    

Data Repository Service
: \<baseUrl>/dms/drs
: Data Repository
    : \<baseUrl>/dms/drs/<repository_id>
    : Sample Repository
        : <baseUrl>/dms/drs/samples
    : XRD Repository
        : <baseUrl>/dms/drs/xrd

Data Catalog Service
: \<baseUrl>/dms/dcs
: Data Catalog
    : \<baseUrl>/dms/dcs/<catalog_id>
