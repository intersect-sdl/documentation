## List of Entities

Campaign
:   Campaign explanation

Campaign Plan
: A campaign plan expresses the steps within the Campaign

Campaign Trace
: A Campaign Trace logs the execution steps of a particular campaign

Experiment
: An experiment is representing a generic experiment within a campaign

Experiment Plan
: An experiment plan contains the steps for the execution of the experiment

Experiment Trace
: An Experiment Trace logs the execution of a particular experiment

Chemspeed Program
: The program to be executed 

Chemspeed Recepie
: The recepie for the synthesis 

Chemspeed Process Data
: Internal instrumentation data of the equipment
  Examples include i.e., actual quasntities dispensed, temperature curves of the furnace, etc.

Synthgesis Product, Sample & Subsample
: A digital representation of the product of the synthesis.
  It is used to track the product and varius samaples during synthesis and analysis

Analysis Results
: The results of the different analisis steps.
  - XRD Data
  - PLC Data
  - TCA MS Data
  - Reaction Data (Series of images)

Asset
: anything stored in a storage backend

Observation
: The act of observing a Feature of interest
